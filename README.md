# single

注意：此文件Single.h可以用于ARC环境和MRC环境中

使用方法（提供的Tool文件中有添加代码样例）：

1.在需要实现单例的类中导入头文件  Single.h

2.在需要的地方添加如下代码：

//.h文件

singleH(Tools) //变成+ (instancetype)shareTools;，括号中的名字你可以使用你想要的名字。如果使用singleH(User)，就变成+ (instancetype)shareUser;
   
//.m文件

singleM(Tools)
